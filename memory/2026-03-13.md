## 项目任务：卷内文件目录生成
### 毛喜瑞交易咨询项目
- **目标**: 为“毛喜瑞交易咨询项目”创建符合规范的《期货交易咨询业务档案卷内文件目录》Word文档。
- **核心要求**:
  1. 文档第一行：“附件2”，黑体三号，居左。
  2. 文档第二行：“期货交易咨询业务档案卷内文件目录”，黑体小二，居中。
  3. 空一行后，创建一个8列的统一表格。
  4. 表格结构:
     - 第1行: A1单元格为“档号”(宽度=序号列宽)，B1:H1合并单元格填入档号值。
     - 第2行: A2单元格为“案卷题名”(宽度=序号列宽)，B2:H2合并单元格填入案卷题名。
     - 第3行: 
        - A3单元格为“客户名称”(宽度=序号列宽)
        - B3:D3合并单元格填入客户名称
        - E3单元格为“客户代码”(宽度=日期列宽)
        - F3:H3合并单元格填入客户代码
     - 第4行: 标准表头（序号、文件题名、责任者、经办人、日期、材料性质、起止页号、备注）。
     - 第5-12行: 8个文件清单，序号从`0001`开始。
     - 第13行 (小计): 在文件清单末尾添加一行，在B列输入“小计”。
  5. 字体: 表格内文字为宋体小四。
  6. 布局: 文件题名列加宽，整体紧凑美观。
