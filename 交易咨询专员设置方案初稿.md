# 关于在业务单位设置交易咨询专员协助研究所开展交易咨询项目的请示

**申万期货研究所**
**2026年3月**

---

## 一、项目背景与目的

随着公司交易咨询业务的快速发展，为更好地整合公司内部资源，提升交易咨询项目的开发效率和服务质量，现拟在各业务单位（包括但不限于营业部、业务部门、子公司）设置**交易咨询专员**岗位。该岗位由各业务单位中已具备交易咨询资格的员工兼职担任，旨在搭建研究所与一线业务单位的常态化沟通协作机制。

### 1.1 现状分析
- 交易咨询项目开发依赖研究所分析师与营销人员的直接对接，沟通链条较长
- 营销人员对交易咨询服务的理解深度不一，项目转化率有待提升
- 研究所产能有限，无法实时响应所有业务单位的需求
- 中后台具备交易咨询资格的员工资源尚未充分挖掘

### 1.2 设置意义
1. **强化业务渗透**：建立覆盖全公司的交易咨询服务网络
2. **提升转化效率**：专人专岗推动交易咨询项目落地
3. **优化资源配置**：发挥中后台专业人才的业务价值
4. **增强客户粘性**：提供更精准、更及时的咨询服务

---

## 二、岗位设置方案

### 2.1 配置原则
- **覆盖范围**：各业务单位（营业部、业务部门、子公司）
- **配备数量**：每单位1-2名，根据业务规模适度调整
- **人员要求**：
  - 已取得公司交易咨询资格认证
  - 熟悉本单位业务特点与客户结构
  - 具备良好的沟通协调能力
  - 优先考虑中后台专业人员

### 2.2 选拔流程
1. **单位推荐**：各业务单位根据人员情况推荐合适人选
2. **资格审核**：研究所会同人力资源部审核资格条件
3. **集中培训**：通过审核人员参加统一岗前培训
4. **正式聘任**：颁发聘任证书，纳入专员管理体系

---

## 三、岗位职责

交易咨询专员作为研究所与业务单位的**桥梁**和**催化剂**，主要承担以下职责：

### 3.1 宣导推广职责
1. **内部宣导**：定期向本单位营销人员宣讲解读最新的交易咨询产品、服务模式及成功案例，确保一线人员清晰掌握交易咨询的业务价值
2. **意识培养**：在日常工作中，持续提醒营销人员关注客户潜在的交易咨询需求，将交易咨询思维融入日常客户服务
3. **信息传递**：及时将研究所发布的研究观点、策略报告、市场分析等专业内容，转化为营销人员易于理解的语言进行内部传播

### 3.2 需求对接职责
1. **需求收集**：主动收集本单位客户（尤其是机构客户、高净值客户）的交易咨询需求，进行初步筛选和分类
2. **项目发掘**：协助营销人员识别潜在的交易咨询项目机会，提供专业判断支持
3. **桥梁作用**：在营销人员与研究所之间建立高效沟通渠道，确保需求准确传递

### 3.3 服务支持职责
1. **项目跟进**：参与交易咨询项目的初期接洽，协助营销人员明确客户需求
2. **信息传递**：在项目立项后，负责将客户的具体需求及时、准确地传递给研究所项目负责人
3. **协调沟通**：协助解决项目实施过程中的沟通问题，确保项目顺利推进

### 3.4 质量反馈职责
1. **服务反馈**：收集客户和营销人员对交易咨询服务的反馈意见
2. **效果评估**：协助研究所对项目效果进行跟踪评估
3. **持续改进**：基于反馈提出服务优化建议

## 四、服务提供模式

### 4.1 组织架构
```
客户需求 → 营销人员 → 交易咨询专员 → 研究所项目组 → 服务交付
              ↑           ↑              ↑
          日常对接  专业桥梁  项目执行
```

### 4.2 具体服务流程

#### 阶段一：项目开发期
1. **需求识别**：营销人员在客户服务中发现潜在交易咨询需求
2. **专员介入**：营销人员与本单位交易咨询专员初步沟通
3. **需求梳理**：专员协助营销人员梳理客户核心需求、资金规模、风险偏好等关键信息
4. **初步评估**：专员对需求可行性进行初步判断，提供专业建议

#### 阶段二：项目执行期
1. **建群管理**：项目确定后，研究所指定分析师或服务支持岗担任**项目负责人**，建立企业微信专项服务群
2. **群成员构成**：
   - 研究所：项目负责人、相关分析师（1-2名）
   - 业务单位：交易咨询专员、对接营销人员
   - 客户方：指定联系人（1-2名）
3. **服务内容**：
   - **常规服务**：项目负责人定期发送相关品种的日报、周报等基础研究成果
   - **定制服务**：交易咨询专员根据客户需求，在群内**定向请求**特定品种的策略报告、专题研究
   - **研究报告来源**：相关分析师将研究报告同步至研究所内部研究群，专员可转发至项目群
4. **沟通机制**：
   - 常规沟通：工作日固定时间（如上午10:00）进行日常交流
   - 紧急响应：重大行情波动时，24小时内响应客户紧急需求
   - 定期复盘：每月召开线上项目复盘会，评估服务效果

#### 阶段三：项目维护期
1. **持续跟踪**：专员定期与客户沟通，了解策略执行情况
2. **需求迭代**：根据市场变化和客户需求调整，提出新的服务建议
3. **关系维护**：协助营销人员维护客户关系，挖掘深度合作机会

### 4.3 服务特色
1. **精准对接**：专员熟悉本单位客户特点，能更精准匹配研究所资源
2. **快速响应**：缩短需求传递链条，提升服务响应速度
3. **专业支持**：专员具备专业资格，能提供初步的专业判断
4. **持续服务**：建立长期服务机制，增强客户粘性

## 五、激励方案

### 5.1 激励原则
1. **市场化导向**：激励水平与项目价值、服务难度相匹配
2. **利益共享**：兼顾营销人员、专员、研究所三方利益
3. **可持续性**：确保激励方案长期可行，促进业务持续发展
4. **分级激励**：根据项目复杂度、服务周期、客户规模设置差异化激励

### 5.2 初步激励标准（建议调整）

#### 原方案分析
- **单品种项目**：300元/年 → **偏低**，难以形成有效激励
- **单板块项目**：1000元/年 → **适中偏低**
- **全品种项目**：3000元/年 → **适中**
- **分配比例**：营销50-60%、专员20-30%、研究所20-30% → **需优化**

#### 优化建议方案

**方案一：基础激励标准（推荐）**
| 项目类型 | 服务内容 | 年度激励总额 | 分配建议 | 备注 |
|---------|---------|-------------|---------|------|
| **A类：单品种** | 针对单一品种的策略服务 | **800元** | 营销:400元(50%)<br>专员:240元(30%)<br>研究所:160元(20%) | 原300元过低，调整至合理水平 |
| **B类：单板块** | 覆盖一个板块（如黑色系、农产品） | **1,500元** | 营销:750元(50%)<br>专员:450元(30%)<br>研究所:300元(20%) | 体现板块服务的额外价值 |
| **C类：全品种** | 覆盖多个板块的全品种服务 | **4,000元** | 营销:2,000元(50%)<br>专员:1,200元(30%)<br>研究所:800元(20%) | 鼓励开发高价值客户 |

**方案二：阶梯式激励（备选）**
| 客户资产规模 | 单品种 | 单板块 | 全品种 |
|-------------|--------|--------|--------|
| 500万以下 | 500元 | 1,200元 | 3,000元 |
| 500-1000万 | 800元 | 1,500元 | 4,000元 |
| 1000万以上 | 1,200元 | 2,000元 | 6,000元 |

### 5.3 分配比例优化建议

**推荐分配结构：营销50%、专员30%、研究所20%**

#### 分配逻辑：
1. **营销人员（50%）**：承担客户开发、关系维护的核心职责，应获得主要激励
2. **交易咨询专员（30%）**：提供专业支持、需求对接、服务协调，价值应充分体现
3. **研究所（20%）**：提供研究支持、项目执行、质量控制，保障服务质量

#### 特殊情形处理：
1. **纯维护项目**（由专员独立维护）：专员60%、研究所40%
2. **重大创新项目**：可设立专项奖励，额外奖励创新贡献
3. **跨单位协作项目**：按实际贡献比例分配

### 5.4 发放机制
1. **发放周期**：按季度发放，基于实际服务月份计算
2. **发放依据**：
   - 项目签约确认（营销部分）
   - 服务记录完整（专员部分）
   - 客户满意度达标（研究所部分）
3. **考核机制**：
   - 客户满意度≥85分，全额发放
   - 客户满意度70-84分，发放80%
   - 客户满意度<70分，暂停发放并整改

### 5.5 其他激励措施
1. **年度评优**：设立“金牌交易咨询专员”奖项，给予额外奖金和荣誉
2. **晋升加分**：专员表现纳入个人绩效考核和晋升评价
3. **培训优先**：优秀专员优先参加高端培训、外部交流
4. **资源共享**：专员可优先获得研究所内部研究资源

## 七、预期成效

### 7.1 短期成效（6个月内）
1. **项目数量提升**：预计新增交易咨询项目20-30个
2. **服务响应提速**：需求响应时间缩短50%以上
3. **客户满意度提升**：客户满意度目标提升至90%
4. **资源利用率提高**：中后台专业人员业务参与度显著提升

### 7.2 中长期成效（1-3年）
1. **业务规模扩大**：交易咨询收入年增长率目标30%以上
2. **专业品牌树立**：形成公司特色的交易咨询服务模式
3. **人才梯队建设**：培养一批既懂业务又懂专业的复合型人才
4. **客户粘性增强**：深度绑定一批核心机构客户

## 八、风险控制与保障措施

### 8.1 主要风险
1. **角色冲突风险**：专员本职与兼职工作可能冲突
2. **服务质量风险**：专员专业水平参差不齐影响服务
3. **利益分配风险**：激励分配可能引发内部矛盾
4. **合规风险**：服务过程中可能出现的合规问题

### 8.2 控制措施
1. **明确边界**：清晰界定专员工作职责和时间投入
2. **加强培训**：建立定期培训机制，提升专业能力
3. **透明管理**：激励分配公开透明，建立申诉机制
4. **合规督导**：研究所专人负责合规审查和指导

### 8.3 资源保障
1. **组织保障**：成立专项工作组，由研究所领导牵头
2. **制度保障**：制定配套管理制度和操作流程
3. **系统保障**：优化现有系统，支持专员工作需求
4. **经费保障**：确保激励资金及时足额到位
